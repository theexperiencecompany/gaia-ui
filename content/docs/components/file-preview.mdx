- **Smart Type Detection** - Automatically detects 20+ file types
- **Custom Icons** - Different icons for images, documents, code, media, archives
- **Image Thumbnails** - Shows actual image previews for image files
- **Upload Progress** - Loading spinner for files being uploaded
- **Remove Button** - Hover-reveal delete button with callback
- **File Type Badge** - Shows formatted file type (PDF, DOC, etc.)

## Usage

```tsx
import { FilePreview } from "@/components/ui/file-preview";

export default function Example() {
  const files = [
    {
      id: "1",
      url: "/uploads/document.pdf",
      name: "report.pdf",
      type: "application/pdf",
    },
  ];

  return (
    <FilePreview
      files={files}
      onRemove={(id) => console.log("Remove:", id)}
    />
  );
}
```

## Props

### FilePreview

| Prop | Type | Description |
| --- | --- | --- |
| files | UploadedFile[] | Array of files to display |
| onRemove | (id: string) => void | Callback when remove button is clicked |
| className | string | Additional CSS classes |

### UploadedFile

| Property | Type | Description |
| --- | --- | --- |
| id | string | Unique file identifier |
| url | string | File URL (for image previews) |
| name | string | File name |
| type | string | MIME type (e.g., "image/png") |
| description | string | Optional description |
| isUploading | boolean | Show loading spinner |

## Supported File Types

The component automatically detects and shows appropriate icons for:

- **Images** - jpg, png, gif, webp, svg
- **Documents** - pdf, doc, docx, odt, rtf
- **Spreadsheets** - xls, xlsx, csv, ods
- **Code** - js, ts, jsx, tsx, py, java, html, css, json, yaml
- **Video** - mp4, avi, mov, mkv
- **Audio** - mp3, wav, ogg
- **Archives** - zip, rar, tar, gz, 7z

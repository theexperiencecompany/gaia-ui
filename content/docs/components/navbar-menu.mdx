## Basic Usage

A simple navbar menu with basic links and descriptions.

<ComponentPreview
  code={`const basicSections = [
  {
    id: "pages",
    links: [
      {
        label: "About",
        href: "/about",
        description: "Learn more about our company",
      },
      {
        label: "Blog",
        href: "/blog",
        description: "Read our latest articles",
      },
      {
        label: "Contact",
        href: "/contact",
        description: "Get in touch with us",
      },
      {
        label: "Careers",
        href: "/careers",
        description: "Join our growing team",
      },
    ],
  },
];

<div className="relative h-[300px] w-full">
  <NavbarMenu activeMenu="pages" sections={basicSections} />
</div>`}
>
  {(() => {
    const basicSections = [
      {
        id: "pages",
        links: [
          {
            label: "About",
            href: "/about",
            description: "Learn more about our company",
          },
          {
            label: "Blog",
            href: "/blog",
            description: "Read our latest articles",
          },
          {
            label: "Contact",
            href: "/contact",
            description: "Get in touch with us",
          },
          {
            label: "Careers",
            href: "/careers",
            description: "Join our growing team",
          },
        ],
      },
    ];
    return (
      <div className="relative h-[300px] w-full">
        <NavbarMenu activeMenu="pages" sections={basicSections} />
      </div>
    );
  })()}
</ComponentPreview>

## Usage

```tsx
import { NavbarMenu } from "@/components/ui/navbar-menu";

export default function Example() {
  const [activeMenu, setActiveMenu] = useState<string | null>(null);

  const sections = [
    {
      id: "product",
      links: [
        {
          label: "Features",
          href: "/features",
          description: "Explore our features",
          icon: <FeatureIcon />,
        },
        // ... more links
      ],
    },
  ];

  return (
    <div className="relative">
      <button onClick={() => setActiveMenu("product")}>Product</button>
      {activeMenu && (
        <NavbarMenu
          activeMenu={activeMenu}
          sections={sections}
          onClose={() => setActiveMenu(null)}
        />
      )}
    </div>
  );
}
```

## Props

### NavbarMenu

| Prop       | Type                | Default   | Description                                 |
| ---------- | ------------------- | --------- | ------------------------------------------- |
| activeMenu | string              | -         | The ID of the currently active menu section |
| sections   | NavbarMenuSection[] | -         | Array of menu sections with links           |
| onClose    | () => void          | undefined | Optional callback when menu should close    |

### NavbarMenuSection

| Property   | Type             | Description                                              |
| ---------- | ---------------- | -------------------------------------------------------- |
| id         | string           | Unique identifier for the section                        |
| links      | NavbarMenuLink[] | Array of links in this section                           |
| gridLayout | string           | Optional custom grid layout classes (default: 2 columns) |

### NavbarMenuLink

| Property        | Type            | Description                                        |
| --------------- | --------------- | -------------------------------------------------- |
| label           | string          | Link text                                          |
| href            | string          | Link URL                                           |
| icon            | React.ReactNode | Optional icon component                            |
| external        | boolean         | Whether link opens in new tab                      |
| description     | string          | Optional description text                          |
| backgroundImage | string          | Optional background image URL for featured items   |
| rowSpan         | number          | Optional row span for grid layout (2 for featured) |

## Features

- **Smooth Animations**: Uses Framer Motion for buttery smooth enter/exit animations
- **Flexible Grid Layouts**: Customize grid layouts per section (2-col, 3-col, etc.)
- **Featured Items**: Support for large featured items with background images
- **Icon Support**: Add icons to any menu item
- **External Links**: Automatically handles external links with proper attributes
- **Responsive**: Works on all screen sizes
- **Accessible**: Keyboard navigation and ARIA attributes

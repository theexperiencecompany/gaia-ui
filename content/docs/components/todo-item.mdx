## Features

- **Interactive Checkbox** - Toggle todo completion with visual feedback
- **Priority Levels** - High, medium, low, and none with distinct color coding
- **Due Dates** - Smart date formatting with overdue/today indicators
- **Labels & Tags** - Color-coded labels for categorization
- **Projects** - Link todos to projects with custom colors
- **Subtasks** - Track progress with subtask completion count
- **Click Handlers** - Custom handlers for clicks and completion toggles

## Usage

```tsx
import { TodoItem } from "@/components/ui/todo-item";

export default function Example() {
  return (
    <TodoItem
      id="1"
      title="Complete project documentation"
      description="Write comprehensive docs for the new API"
      completed={false}
      priority="high"
      dueDate={new Date()}
      onToggleComplete={(id, completed) => {
        console.log(`Todo ${id} completed: ${completed}`);
      }}
    />
  );
}
```

## Props

| Prop | Type | Description |
| --- | --- | --- |
| id | string | Unique identifier for the todo |
| title | string | Todo title |
| description | string | Optional description |
| completed | boolean | Completion status |
| priority | "high" \| "medium" \| "low" \| "none" | Priority level |
| dueDate | string \| Date | Optional due date |
| labels | TodoLabel[] | Array of labels |
| subtasks | TodoSubtask[] | Array of subtasks |
| project | TodoProject | Optional project assignment |
| onToggleComplete | (id, completed) => void | Completion toggle handler |
| onClick | (id) => void | Click handler |
| isSelected | boolean | Selected state |
| className | string | Additional CSS classes |
